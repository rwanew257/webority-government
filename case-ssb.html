Project Overview

Security Assessment for a Paramilitary Force

Sashastra Seema Bal is a Central Armed Police Force under the Ministry of Home Affairs, responsible for guarding India’s borders with Nepal and Bhutan. With an expanding digital infrastructure supporting operations, HR, and administrative functions, SSB required a comprehensive security assessment aligned with GoI cybersecurity mandates.

This engagement was conducted under strict confidentiality protocols — all findings, vulnerabilities, and remediation recommendations are classified. The scope covered web applications, network infrastructure, and internal systems.

Scope of Work

What We Delivered

  1. Web Application Security Assessment
    Comprehensive black-box and grey-box testing of SSB’s web-facing applications — covering OWASP Top 10 vulnerabilities, authentication mechanisms, session management, and data exposure risks.
  2. Network Infrastructure Assessment
    Internal and perimeter network security review — firewall configurations, network segmentation, open ports, and protocol vulnerability analysis.
  3. Vulnerability Classification & Prioritisation
    All identified vulnerabilities classified by severity (Critical / High / Medium / Low) with CVSS scoring, business impact assessment, and exploitability ratings.
  4. CERT-In Compliance Review
    Gap analysis against CERT-In cybersecurity guidelines and MHA security directives — identifying non-compliance areas and required remediation steps.
  5. Formal VAPT Report
    Comprehensive security assessment report with executive summary for leadership, detailed technical findings for IT teams, and a prioritised remediation roadmap with timelines.
  6. Remediation Guidance & Re-testing
    Post-remediation guidance for SSB’s IT team and re-testing of critical vulnerabilities to confirm successful closure.
